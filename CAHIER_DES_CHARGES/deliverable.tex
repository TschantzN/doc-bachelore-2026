\section{Délivrables du projet}
À l'issue du travail, les livrables suivants seront fournis :

\begin{itemize}
    \item \textbf{Livrables Techniques :}
    \begin{itemize}
        \item Dépôts Git contenant le code source modifié du pilote Morse Micro (MM8108) ainsi que les 
        directives de compilation.
        \item Code source de l'application de réception développée sous FreeRTOS (basé sur le \textit{mm-iot-sdk}).
        \item Ensemble des ressources de configuration (scripts ou fichiers de paramétrage) permettant 
        la reproduction de l'environnement réseau de l'émetteur et du récépteur.
    \end{itemize}
    
    \item \textbf{Livrables Documentaires (intégrés au rapport final) :}
    \begin{itemize}
        \item \textbf{Rapport final :} Document central incluant l'analyse du pilote, les modifications effectuées 
        pour le forçage du LDPC/MCS, ainsi que les protocoles et résultats des mesures de performance (latence, portée).
        \item \textbf{Guide de mise en œuvre :} Documentation technique détaillant les étapes d'installation, de 
        configuration et de déploiement du système complet.
        \item \textbf{Journal de bord :} Document retraçant chronologiquement les étapes de développement, les
        défis techniques rencontrés et les décisions prises durant le projet.
    \end{itemize}
    
    \item \textbf{Livrables pour la soutenance :}
    \begin{itemize}
        \item Poster scientifique synthétisant les enjeux et résultats du projet.
        \item Support de présentation pour la défense orale devant le jury.
    \end{itemize}
\end{itemize}
